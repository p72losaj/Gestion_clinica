# Gestion_clinica
Desarrollo de un software de gestión para una clinica medica

Datos del cliente:
  · Nombre
  · Apellidos
  · Fecha nacimiento
  · Publico o privado
  · Direccion
  · Telefono

Datos de la cita:
  · Hora
  · Fecha
  · Motivo de la cita
  - No pueden pisarse

Datos tratamiento:
  · Dia del comienzo
  · Dia de finalización
  · Tratamiento

Historial medico:
  - No se puede modificar.

Acceso público sin restricciones

Funciones:
  · Con ventanitas
  · Registro usuario
  · Buscar usuarios (mostrar lista)
  · Calendario con las citas
  · Editar datos usuario
  · Cancelación de tratamiento
  · Añadir citas
  - Linux
